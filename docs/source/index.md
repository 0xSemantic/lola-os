# LOLA OS Documentation

## Welcome to LOLA OS
LOLA OS is an open-source framework for building sovereign on-chain AI agents. It bridges AI reasoning (via LLMs) and blockchain execution (EVM-compatible chains), enabling developers to create autonomous agents that read, reason, and act on the blockchain and web.

### Who Is This For?
- **Beginners**: New to AI or blockchain? Start with Quickstart to set up and run your first agent in minutes.
- **Intermediate Users**: Explore Concepts and Tutorials to build custom agents with tools and workflows.
- **Advanced Users**: Dive into API Reference and Integration Guide for extending LOLA OS with multi-chain support, custom tools, and production deployments.

### Key Features
- EVM-Native Blockchain Integration: Full read/write on Ethereum, Polygon, Arbitrum, Base via web3.py.
- AI Orchestration: Extend LangGraph for workflows, LiteLLM for provider-agnostic LLM routing (default Gemini free tier).
- Web Research: Crawl4AI for structured extraction from dynamic sites.
- CLI for Scaffolding/Execution: Quick start with `lola create/run`.
- Modular Design: Tools, chains, agents, and libs for customization without lock-in.
